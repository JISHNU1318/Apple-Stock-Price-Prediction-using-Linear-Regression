# Apple Stock Price Prediction using Linear Regression

This project is a beginner Machine Learning application built using Python and Scikit-learn. The model uses Linear Regression to analyze the relationship between Apple stock opening and closing prices.

## Features
- Data preprocessing and cleaning
- Handling missing and invalid values
- Train-test split for model evaluation
- Linear Regression model training
- Prediction of stock closing prices
- Performance evaluation using MSE and R² Score
- Data visualization using Matplotlib

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Load stock market dataset
2. Clean and preprocess data
3. Select features and target variables
4. Train Linear Regression model
5. Predict stock closing prices
6. Evaluate model performance
7. Visualize regression results
